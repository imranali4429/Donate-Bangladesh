What is the DOM (Document Object Model)?
The DOM (Document Object Model) is a programming interface for web documents.
The browser takes your HTML and turns it into a tree-like structure called the DOM. Every HTML element, attribute, and text becomes a node (object) in that tree.
With JavaScript, you can use the DOM to read, modify, add, or delete elements, and also handle events (like clicks, inputs, etc.).
